The program can be run in the terminal as such:

python3 ./urban_planning.py [file] [alg]

Where [file] is replaced with the path to the input file.
[alg] is to replaced with HC if you would like to run the hill
climbing / annealing function
[alg] is to be replaced with GA if you would like to run the
genetic algorithm function

An example of the command with values filled in:

python3 ./urban_planning.py test_file.txt GA

The following libraries will need to be installed:
argparse
math
numpy
random
time
